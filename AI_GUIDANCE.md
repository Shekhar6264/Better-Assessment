# AI Usage and Guidance

This project was built using AI-assisted development.

## Constraints Given to AI:
- Keep architecture modular
- Separate models, services, and routes
- Avoid business logic inside routes
- Enforce JWT validation on protected routes
- Prevent double seat booking
- Use simple and readable code over clever abstractions

## Review Process:
All generated code was manually reviewed and tested.
Business logic was validated using automated tests.

AI was used as an assistant, not as an autonomous coder.
